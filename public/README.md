# Public Assets

## Required Logo
Add your crown logo as `kk-logo.png` in this directory.

**Requirements:**
- File name: `kk-logo.png`
- Recommended size: 512x512px or larger
- Format: PNG with transparency
- Should be a crown icon for the "Built for Kings" branding

This logo will appear in:
- Header component (top-left)
- Browser favicon/tab icon
